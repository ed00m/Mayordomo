Crear direcciones IP virtuales en GNU/Linux CentOS 5

Posted on octubre 9, 2011 by lsilva

Hola comunidad, acá les dejo este breve manual para crear direcciones IP virtuales en GNU/Linux CentOS. Primeramente se visualiza con el comando ifconfig la dirección IP actual en la interfaz de red eth0;

Ahora con la siguiente sintaxis se crea la siguiente dirección IP virtual 192.168.1.13/24 en la interfaz de red eth0:0;

ifconfig eth0:0 192.168.1.13 netmask 255.255.255.0

Para finalizar y que no se borre la configuración de la dirección IP virtual cada vez que se reinicie el servicio de red, se crea un script de nombre ifcfg-eth0:0 en el directorio /etc/sysconfig/network-scripts

touch /etc/sysconfig/network-scripts/ifcfg-eth0:0

Con un editor de texto se ingresa en el archivo ifcfg-eth0:0 los siguientes parámetros, respetando las minúsculas y mayúsculas;

DEVICE=eth0:0

IPADDR=192.168.1.13

NETMASK=255.255.255.0

Listo, puede reiniciar el servicio de red del servidor y la dirección IP virtual no se borrara de la configuración del sistema.

Espero haya sido de utilidad.

Saludos.
